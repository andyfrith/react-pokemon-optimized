## Getting Started

### Prerequisites

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/andyfrith/react-pokemon-optimized.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the dev script
   ```sh
   npm run dev
   ```
